# parking_occupancy_camera

This repository contains the code for the parking occupancy camera project.

## Installation

1. Ensure you have Python 3.9.25 installed on your system.
2. Install the required dependencies by running: 
    pip install -r requirements.txt

## Usage

- To train model use Car Parking System Using Deep Learning.ipynb
- To run the application locally use test.py
- To run the application as a web application use main.py
